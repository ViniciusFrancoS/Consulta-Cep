## 🧰 Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
- [API ViaCEP](https://viacep.com.br)

## 💡 Funcionalidades
- Busca de endereço pelo CEP
- Exibição de logradouro, bairro, cidade e estado
- Design responsivo e moderno
